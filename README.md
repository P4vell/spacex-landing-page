## Setup

To run this project, install it locally using npm:

```bash
# Clone this repository
$ git clone https://github.com/P4vell/spacex-landing-page.git

# Go into the repository
$ cd spacex-landing-page

# Install dependencies
$ npm install

# Run the app
$ npm run dev
```
